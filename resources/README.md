# Native app assets

`icon.png` (1024×1024) is generated by `pnpm run gen:icons`.

After adding or refreshing icons, sync native launcher icons:

```bash
pnpm add -D @capacitor/assets
npx capacitor-assets generate --iconBackgroundColor '#08050c' --splashBackgroundColor '#08050c'
```

Run from the repo root with `android/` or `ios/` present (`pnpm run cap:add:android` / `cap:add:ios`).
